# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.3.0] - 2025-04-10

### Added
- Add usage report generation via `UsageReport` in `usage_report.py`
- Add token-aware rate limiting via `RateLimiter` in `rate_limiter.py`

## [0.2.0] - 2026-04-10

### Added
- Add streaming cost tracking for async LLM calls
- Add cost comparison charts (ASCII bar, SVG, table)
- Add examples/ directory with 4 usage scripts

## [0.1.0] - 2026-04-10

### Added
- Initial release: universal LLM token counting and cost management
